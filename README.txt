Bonjour !

Pour utiliser notre application, il faut lancer le projet unity "Client" avec Unity 2021.3.11f1.

Il est également possible de le lancer en build mais étant donné que c'est super lourd on a pas 
mis de build sur le git. Le code du client est dans ExamReseau\Client\Assets\Scripts

le dossier "Server" contient un projet visual studio à lancer pour que l'application fonctionne.
Le code serveur est dans ExamReseau\Server\Server.
